Detailed Description

template<unicode_input FromEncoding, unicode_char_type ToEncoding, std::ranges::input_range View>
class icubaby::ranges::transcode_view< FromEncoding, ToEncoding, View >::iterator::state

The state class is responsible for transforming the input values to output code units.

It maintains a code point's worth of output code units in its internal buffer. These may be accessed by the owning iterator using the front() and advance() member functions. Once the buffer is empty, it is refilled using the fill() function.
