Detailed Description

A C++ Baby Library to Immediately Convert Unicode. A header only, dependency free, library for C++ 17 or later. Fast, minimal, and easy to use for converting a sequence in any of UTF-8, UTF-16, or UTF-32.

#define ICUBABY_CONCEPT_OUTPUT_ITERATOR ( x)

Defined as std::output_iterator\<x\> if concepts are supported and as typename otherwise.

Used as a shortcut to restrict a specific template argument to be an output iterator in C++ 20 and simply declaring a typename in C++ 17.
