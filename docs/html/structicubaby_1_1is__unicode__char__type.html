Detailed Description

template<typename T>
struct icubaby::is_unicode_char_type< T >

Checks whether the argument is one of the unicode character types.

Provides the constant value which is equal to true, if T is one of the unicode character types as defined by icubaby::character_types. Otherwise, value is equal to false.
